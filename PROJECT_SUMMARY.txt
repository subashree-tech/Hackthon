🚀 QUICK START (Choose One)
================================================================================

OPTION 1: AUTOMATED SETUP (Recommended)
----------------------------------------
Windows:
  1. Double-click setup.bat
  2. Edit .env with your API keys
  3. Run: venv\Scripts\activate
  4. Run: python run.py
  5. Open: http://localhost:5000

Mac/Linux:
  1. Run: chmod +x setup.sh && ./setup.sh
  2. Edit .env with your API keys
  3. Run: source venv/bin/activate
  4. Run: python run.py
  5. Open: http://localhost:5000

OPTION 2: MANUAL SETUP
-----------------------
  1. python -m venv venv
  2. Activate venv (see commands above)
  3. pip install -r requirements.txt
  4. Copy env.example to .env
  5. Edit .env with API keys
  6. python run.py

================================================================================
🔑 REQUIRED API KEYS
================================================================================

1. STABILITY AI (Required for AI generation)
   - Get: https://platform.stability.ai/
   - Free trial available
   - Add to .env: STABILITY_API_KEY=sk-...

2. OPENAI (Optional, for better voice)
   - Get: https://platform.openai.com/
   - Add to .env: OPENAI_API_KEY=sk-...
   - If skipped: Uses free Google Speech Recognition

3. SECRET KEY (Required)
   - Add to .env: SECRET_KEY=any-random-string
